##  Project Description

This project is a real-time collaborative document editor that allows multiple users to work on a shared document simultaneously. It mimics functionality similar to Google Docsâ€”users can see live edits and collaborate in real-time without needing to reload the page.

##  What the App Does

- Multiple users can **edit a single document** at the same time.
- Changes made by one user are **instantly reflected** on all other users' screens.
- Supports **basic text editing** in real-time.
- Provides a **live sync** experience with a responsive user interface.
